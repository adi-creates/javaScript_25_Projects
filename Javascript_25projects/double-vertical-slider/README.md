# Double Vertical Slider

A simple, responsive double vertical slider built with HTML, CSS, and JavaScript. This project showcases an interactive UI component where two vertical sliding panels move in opposite directions, creating an eye-catching effect perfect for portfolios, presentations, or landing pages.

## Demo

Demo Link: https://adi-creates.github.io/double-vertical-slider/

## Features

- **Dual Opposing Sliders**: Each panel slides vertically in opposite directions.
- **Smooth Transitions**: Beautiful CSS-based transitions for a polished user experience.
- **Responsive Design**: Optimized for various screen sizes, including mobile and desktop.
- **Customizable Content**: Easily modify slides with your own content or images.

## Technologies Used

- **HTML** (44.9%)
- **CSS** (31.8%)
- **JavaScript** (23.3%)

## Getting Started

### Prerequisites

- A modern web browser

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/adi-creates/double-vertical-slider.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd double-vertical-slider
   ```

3. **Open `index.html` in your browser:**
   - Double-click `index.html` or
   - Run this command:
     ```bash
     open index.html    # On macOS
     start index.html   # On Windows
     xdg-open index.html  # On Linux
     ```

No build steps or dependencies required!

---

## Usage

- **Navigate Slides:** Use on-screen arrows or swipe (on mobile) to slide up and down.
- **Add Your Content:** Edit the HTML and CSS files to insert your own images, text, or content blocks.

---

## Customization

- **Add/Remove Slides:** In `index.html`, duplicate or remove slide sections as needed.
- **Colors & Styles:** Update styles in `style.css` for personalized themes and transitions.
- **JavaScript Controls:** Enhance functionality in `script.js` for advanced use cases.
